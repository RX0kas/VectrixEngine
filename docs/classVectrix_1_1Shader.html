Detailed Description

This class represent a shader.

This class represent a shader that can be created with Vectrix::ShaderManager::createShader()

◆ bind()

void Vectrix::Shader::bind ( ) const

Tell the GPU that we need to use this shader
This function tell the GPU that this shader will be used for all subsequent draw commands until another one is bound.

Warning
Only one shader can be bound at a time, if you call bind on another shader it will stop using this one
The shader is automatically unbound at each new frame
